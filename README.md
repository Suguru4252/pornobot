QuickChat Messenger

Современный веб-мессенджер с реальным временным общением, похожий на Telegram. Полностью готов к развертыванию и использованию.

🚀 Возможности

· ✅ Регистрация и авторизация пользователей

· ✅ Общение в реальном времени через WebSocket

· ✅ Групповые и личные чаты

· ✅ Общий чат для всех пользователей

· ✅ Профиль пользователя с редактированием

· ✅ Поиск по чатам

· ✅ Статусы онлайн/офлайн

· ✅ Адаптивный дизайн для мобильных устройств

· ✅ Безопасность с JWT токенами


🛠 Технологии

· Frontend: HTML5, CSS3, Vanilla JavaScript
· Backend: Node.js, Express.js
· Real-time: WebSocket (ws)
· Аутентификация: JWT, bcrypt
· Хостинг: Готово для Render.com

📦 Быстрый старт

1. Клонирование репозитория

```bash
git clone https://github.com/your-username/quickchat-messenger.git
cd quickchat-messenger
```

2. Установка зависимостей

```bash
npm install
```

3. Запуск в разработке

```bash
npm run dev
```

4. Запуск в продакшене

```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

🌐 Деплой на Render.com

1. Подготовка репозитория

Убедитесь, что все файлы закоммичены и отправлены в GitHub:

```bash
git add .
git commit -m "Deploy ready"
git push origin main
```

2. Создание сервиса на Render

1. Перейдите на Render.com
2. Нажмите "New +" → "Web Service"
3. Подключите ваш GitHub репозиторий
4. Заполните настройки:

Build Settings:

· Build Command: npm install
· Start Command: npm start

Environment Variables:

```env
NODE_ENV=production
JWT_SECRET=your-super-secret-jwt-key-here
```

1. Нажмите "Create Web Service"

3. Домен

Render автоматически предоставит домен вида:
https://your-app-name.onrender.com

🔧 API Endpoints

Аутентификация

· POST /api/auth/register - Регистрация
· POST /api/auth/login - Авторизация

Пользователи

· GET /api/users/profile - Получить профиль
· PUT /api/users/profile - Обновить профиль

Чаты

· GET /api/chats - Получить список чатов
· POST /api/chats - Создать чат/группу
· GET /api/chats/:id/messages - Получить сообщения
· POST /api/chats/:id/messages - Отправить сообщение

WebSocket Events

· authenticate - Аутентификация
· send_message - Отправка сообщения
· new_message - Новое сообщение
· user_online - Пользователь онлайн
· user_offline - Пользователь офлайн
