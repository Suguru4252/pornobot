# 🔞 Telegram Adult Chat Bot

Бот для взрослых с выбором персонажа и "ИИ" ответами на основе ключевых слов.

## Возможности
- Выбор развратной девушки или пошлого парня
- Контекстные ответы на сообщения
- Поддержка ключевых слов
- Простое расширение словарей

## Установка
1. Клонируй репозиторий
2. Установи зависимости: `pip install pyTelegramBotAPI`
3. Замени токен в `bot.py`
4. Запусти: `python bot.py`

## Команды
- `/start` - Начать и выбрать персонажа
- `/change` - Сменить персонажа
- `/help` - Помощь

## Лицензия
Только для лиц старше 18 лет.
